# FleetCheck 🚗

Aplicación móvil para la gestión digital de checklists de inspección de vehículos empresariales.

## Objetivos

- Eliminar el uso de papel.
- Registrar inspecciones de vehículos.
- Reportar incidencias con fotografías.
- Llevar historial de revisiones.
- Facilitar el seguimiento de la flota.

## Tecnologías

- React Native (Expo)
- Node.js
- Express
- MongoDB
- React
- JWT
