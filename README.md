# 🚗 SEGEA Control

Sistema de Gestión de Inspecciones y Administración de Vehículos desarrollado con React Native, Expo y TypeScript.

## 📱 Descripción

SEGEA Control es una aplicación móvil diseñada para facilitar la administración de vehículos de una empresa, permitiendo registrar, visualizar, editar y eliminar vehículos de manera sencilla mediante una interfaz moderna e intuitiva.

---

## ✨ Funcionalidades

- 🚗 Listado de vehículos
- ➕ Registro de nuevos vehículos
- ✏️ Edición de información
- 🗑️ Eliminación de vehículos
- 🔍 Visualización de detalles
- 📅 Gestión de revisión técnica
- 🔧 Gestión de mantenciones
- 📋 Registro de observaciones
- 🚦 Estado del vehículo (Activo, Mantenimiento, Fuera de servicio)
- 🔤 Formato automático de patentes chilenas
