# FleetCheck

## Descripción

FleetCheck es una aplicación móvil diseñada para digitalizar el proceso de inspección de vehículos de una empresa, reemplazando completamente el uso de formularios en papel.

La aplicación permitirá a los trabajadores realizar inspecciones diarias de las camionetas mediante un checklist digital, registrar observaciones, adjuntar fotografías de fallas y mantener un historial completo de las revisiones.

---

## Objetivo general

Desarrollar una aplicación móvil que facilite el control del estado de los vehículos de la empresa, mejorando la trazabilidad de las inspecciones, reduciendo el uso de papel y permitiendo una gestión eficiente de las incidencias detectadas.

---

## Objetivos específicos

- Digitalizar el checklist diario.
- Registrar evidencias fotográficas.
- Mantener un historial de inspecciones.
- Reducir el uso de papel.
- Facilitar el seguimiento de fallas.
- Mejorar la comunicación entre trabajadores y supervisores.

---

## Usuarios

### Trabajador

- Iniciar sesión.
- Seleccionar una camioneta.
- Completar checklist.
- Adjuntar fotografías.
- Agregar observaciones.
- Enviar revisión.

### Supervisor

- Revisar inspecciones.
- Visualizar fotografías.
- Consultar historial.
- Administrar camionetas.
- Administrar trabajadores.

### Administrador

- Administrar usuarios.
- Configurar checklists.
- Administrar permisos.
